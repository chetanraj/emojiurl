### How to use

Add the emoji.js file path to the of your document head:

```js
<script src="path/to/emoji.js">
```

then call

```js
  URLEmoji.addEmojiToURL();
```
> Everytime you reload a new smiley emoji appears on the url ðŸ˜‰
